>### **1. `sort` 函数的基本作用**
>
>`sort` 是 C++ 标准库 `<algorithm>` 中的一个函数，用于对数组或容器中的元素进行**快速排序**（默认升序）。
> 在图中代码里：
>
>```
>sort(a + 1, a + 1 + n);  // 对数组a的下标1到n的元素排序
>```
>
>------
>
>### **2. 参数解析**
>
>`sort` 需要两个参数，表示要排序的**范围**（左闭右开区间）：
>
>| 参数           | 含义                            | 图中代码解析                 |
>| -------------- | ------------------------------- | ---------------------------- |
>| **第一个参数** | 排序起始位置的指针/迭代器       | `a + 1` → 从 `a[1]` 开始     |
>| **第二个参数** | 排序结束位置的后一位指针/迭代器 | `a + 1 + n` → 到 `a[n]` 结束 |
>
>**为什么是 `a + 1 + n`？**
>
>- C++ 中区间通常是 **[begin, end)**（包含 `begin`，不包含 `end`）
>- 因此 `a + 1 + n` 实际指向 `a[n]` 的下一个位置
>
>------
>
>### **3. 排序范围示意图**
>
>假设 `n = 5`，数组初始状态：
>
>```
>索引: 0   1   2   3   4   5   6
>值:   ?   3   1   4   2   5   ?
>```
>
>- `sort(a + 1, a + 6)` → 排序 `a[1]` 到 `a[5]`
>- 排序后：
>
>```
>索引: 0   1   2   3   4   5   6
>值:   ?   1   2   3   4   5   ?
>```
>
>------
>
>### **4. 输出第 m 大的数**
>
>代码中 `cout << a[n - m + 1] << endl;` 的逻辑：
>
>- 排序后数组是升序，`a[n]` 是最大值
>- 倒数第 `m` 大的数 → `a[n - m + 1]`
>	 （例如 `m=2` 时，输出 `a[5-2+1] = a[4]`）
>
>------
>
>### **5. 常见用法扩展**
>
>#### **(1) 默认升序排序**
>
>```
>int arr[] = {5, 2, 9, 1};
>sort(arr, arr + 4);  // 排序后：1, 2, 5, 9
>```
>
>#### **(2) 降序排序**
>
>```
>sort(arr, arr + 4, greater<int>());  // 排序后：9, 5, 2, 1
>```
>
>#### **(3) 对 vector 排序**
>
>```
>vector<int> v = {5, 2, 9, 1};
>sort(v.begin(), v.end());  // 使用迭代器
>```
>
>#### **(4) 自定义排序规则**
>
>```
>bool cmp(int a, int b) { return a > b; }
>sort(arr, arr + 4, cmp);  // 降序
>```
>
>------
>
>### **6. 注意事项**
>
>1. **下标从 0 还是 1 开始？**
>	- 图中代码从 `a[1]` 开始存数据（竞赛常见写法）
>	- 通常建议从 `a[0]` 开始以兼容标准库函数
>2. **时间复杂度**：
>	 `sort` 平均复杂度是 ​**O(n log n)​**，适合大多数场景
>3. **安全边界**：
>	 确保排序范围不越界（如 `n` 不能超过数组长度）
>
>------
>
>### **7. 完整代码解析（图中例子）**
>
>```
>#include <bits/stdc++.h>
>using namespace std;
>const int N = 1e3 + 7;  // 定义数组最大长度
>int n, m, a[N];         // 全局数组，默认初始化为0
>
>int main() {
>    cin >> n >> m;       // 输入元素个数n和要找的第m大数
>    for (int i = 1; i <= n; i++) 
>        cin >> a[i];     // 输入数据存入a[1]~a[n]
>    
>    sort(a + 1, a + 1 + n); // 对a[1]~a[n]排序
>    
>    cout << a[n - m + 1] << endl; // 输出第m大的数
>    return 0;
>}
>```
>
>------
>

>
>- `sort(起始位置, 结束位置)` 是排序的核心

>- 理解 **[begin, end)** 区间范围是关键
>- 通过调整参数可以实现部分排序或自定义排序
>
>如果你是刚学习排序，建议先用小数组（如n=5）手动模拟排序过程，能更快理解指针移动和区间概念！

##  **为什么在 `#define int long long` 时要用 `signed main()`？**

### **问题背景**

在算法竞赛中，有人会写：

```cpp
#define int long long  // 把所有 int 替换成 long long
```

但如果直接写 `int main()`，它会被替换成 `long long main()`，导致编译错误：

```cpp
int main() { return 0; }  // 被替换成 long long main()，编译错误！
```

因为 C/C++ 标准规定 `main()` 必须返回 `int`，不能是 `long long`。

### **解决方案**

用 `signed` 代替 `int`：

```cpp
signed main() { return 0; }  // 正确，signed 不会被宏替换
```

- `signed` 是 `signed int` 的缩写，符合 `main()` 的返回类型要求。
- 宏 `#define int long long` 不会影响 `signed`。

`%.10lg` 是 C 语言 `printf` 函数中的一个**格式化输出控制符**，用于智能输出浮点数。下面详细解释它的工作原理和与你的代码的关系：

------

### **1. `%.10lg` 的分解说明**

- **`%`**：格式说明符起始标志

- **`.10`**：指定**最大保留10位有效数字**（不是小数点后10位！）

- **`l`**：表示参数是 `double` 类型（`%g` 默认对应 `float`，`%lg` 对应 `double`）

- **

	```
	g
	```

	**：智能选择输出格式：

	- 当指数小于-4或大于等于精度（这里是10）时，使用科学计数法（如 `1.23456E+05`）
	- 否则使用常规小数格式（如 `123.456`）
	- **自动删除末尾无效的零**（如 `1.200` → `1.2`）
	- **自动省略无意义的小数点**（如 `1.0` → `1`）

```c++
#include<bits/stdc++.h> 
using namespace std;
struct Node{
  int val;
  Node *next;
};
int main(){
  int n;
  cin>>n;
  Node Head;
  Node* now=&Head;
  int tmp;
  for(int i=0;i<n;i++){
    cin>>tmp;
    now->next=new Node;
    now=now->next;
    now->val=tmp;
  }
  int m;
  cin>>m;
  while(m--){
    now=&Head;
    cin>>tmp;
    for(int i=0;i<tmp-1;i++){
      now=now->next;
    }
    now->next=now->next->next;
  }
  now=&Head;
  while(now->next!=NULL){
    now=now->next;
    cout<<now->val<<" ";
  }
  return 0;
}
```
